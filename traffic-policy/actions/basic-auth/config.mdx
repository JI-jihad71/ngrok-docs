## Configuration Reference

This is the [Traffic Policy](/docs/http/traffic-policy/) configuration
reference for this action.

### Action Type

`basic-auth`

### Configuration Fields

| Parameter     | Type       | Description                                                                                                                      |
| ------------- | ---------- | -------------------------------------------------------------------------------------------------------------------------------- |
| `credentials` | `string[]` | A list of allowed `username:password` credential pairs. Passwords must be at least 8 characters and no more than 128 characters. |
| `realm`       | `string`   | The HTTP realm of the request as per RFC 7235, defaults to 'ngrok'.                                                              |
| `enforce`     | `bool`     | When set to `false`, all requests are allowed even if basic authentication fails. Defaults to `true`.                            |

### Supported Phases

- `on_http_request`

### Supported Schemes

- `https`
- `http`
